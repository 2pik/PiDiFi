# Changelog

Все заметные изменения в проекте PDF to PPTX Converter будут задокументированы в этом файле.

## [2.0.0] - 2024

### Добавлено
- **Пакетная обработка**: Возможность выбора нескольких PDF файлов для одновременной конвертации
- **Настройка качества рендеринга**: Выбор из 4 уровней (Низкое, Среднее, Высокое, Максимальное)
- **Выбор формата слайда**: Поддержка форматов 16:9, 4:3, A4
- **Система логирования**: Запись событий в лог-файл с поддержкой разных уровней
- **Валидация PDF файлов**: Проверка на существование, размер, защиту паролем и повреждение
- **Ограничение размера файла**: Предупреждение при превышении лимита 100 МБ
- **Индикатор прогресса с ETA**: Отображение оставшегося времени конвертации
- **Оптимизация изображений**: Автоматическое сжатие JPEG изображений (качество 85%)
- **Улучшенная очистка памяти**: Принудительный сборщик мусора после обработки каждой страницы
- **Drag & Drop**: Базовая поддержка перетаскивания файлов
- **Кнопка очистки**: Быстрая очистка списка выбранных файлов

### Изменено
- Увеличен размер окна приложения (650x550)
- Улучшены сообщения об ошибках с указанием возможных причин
- Расширен интерфейс для поддержки новых функций

### Исправлено
- Утечки памяти при обработке больших файлов
- Проблемы с кодировкой при чтении некоторых PDF файлов

### Техническое
- Добавлены type hints для основных функций
- Рефакторинг кода для улучшения читаемости
- Добавлено подробное логирование всех операций

## [1.0.0] - 2024

### Добавлено
- Первоначальная версия приложения
- Конвертация PDF в PPTX с сохранением текста и изображений
- Графический интерфейс в стиле macOS
- Локальная обработка без отправки данных в облако
