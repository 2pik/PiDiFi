На macOS просто выполните в терминале:

1. ✅ Автоматически проверятся и установятся все зависимости (PyMuPDF, python-pptx, Pillow)

## 💿 Установка приложения

После выполнения скрипта:

1. Откройте файл `dist/PDFToPPTXConverter.dmg`
2. Перетащите иконку `PDFToPPTXConverter.app` в папку **Applications** (Программы)
3. Запустите приложение из Launchpad или папки Программы

## 🎯 Как использовать приложение

1. **Запустите приложение** из папки Программы
2. **Нажмите кнопку "Выбрать..."** и выберите один или несколько PDF файлов
3. **Настройте параметры** (качество, формат слайда)
4. **Нажмите "Конвертировать"**
5. **Укажите место сохранения** PPTX файла

## 🔧 Решение проблем

### Ошибка "Не удалось проверить разработчика"
macOS может блокировать приложение от неизвестных разработчиков:

1. Откройте **Системные настройки** → **Защита и безопасность**
2. Нажмите **"Разрешить"** рядом с сообщением о блокировке
3. Или кликните правой кнопкой мыши на приложении → **Открыть** → **Открыть**

### Ошибка при установке зависимостей
Если скрипт не может установить зависимости:

```bash
# Установите вручную
pip3 install PyMuPDF python-pptx Pillow

# Затем запустите сборку снова
./build_dmg.sh
```

### Требуется Python 3
Убедитесь, что установлен Python 3:

```bash
python3 --version
```

Если не установлен:
- Скачайте с [python.org](https://www.python.org/downloads/)
- Или через Homebrew: `brew install python3`
