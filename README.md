# PDF в PPTX Конвертер для macOS

Простое и безопасное приложение для конвертации PDF файлов в формат PowerPoint (PPTX). Работает полностью оффлайн на вашем компьютере.

## ✨ Особенности

- 🔒 **Полная конфиденциальность** — все данные обрабатываются локально, ничего не отправляется в интернет
- 🎨 **Нативный интерфейс** — дизайн в стиле Apple Human Interface Guidelines
- ⚡ **Быстрая конвертация** — оптимизированный алгоритм обработки
- 📄 **Многостраничная поддержка** — конвертирует PDF любой длины
- 🖼️ **Высокое качество** — сохранение оригинального качества изображений
- 🌓 **Тёмная тема** — автоматическая адаптация под системную тему
- 📦 **Офлайн-установка** — работает без интернета после установки зависимостей

## 📥 Установка

### Вариант 1: Готовый установщик (Рекомендуется)

1. Скачайте последнюю версию DMG файла со страницы [Releases](https://github.com/YOUR_USERNAME/pdf-to-pptx-converter/releases)
2. Откройте скачанный файл `PDFToPPTXConverter-v1.0.0.dmg`
3. Перетащите иконку приложения в папку **Applications**
4. Запустите приложение из папки Applications

> **Важно:** При первом запуске macOS может предупредить о неизвестном разработчике. Нажмите правой кнопкой → «Открыть» → «Открыть» в диалоге безопасности.

### Вариант 2: Установка из исходников

```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/pdf-to-pptx-converter.git
cd pdf-to-pptx-converter

# Установите зависимости
pip3 install PyMuPDF python-pptx Pillow

# Запустите приложение
python3 pdf_to_pptx_gui.py
```

### Вариант 3: Создание собственного DMG

```bash
# Сделайте скрипт исполняемым
chmod +x build_dmg.sh

# Создайте установочный DMG
./build_dmg.sh

# DMG файл будет создан в текущей директории
```

> **Примечание:** Команда `hdiutil` доступна только на macOS. На других ОС будет создана только структура `.app` бандла.

## 🚀 Использование

1. Запустите приложение **PDF в PPTX Конвертер**
2. Нажмите кнопку **«Выбрать...»** и укажите PDF файл (или перетащите файл в окно)
3. Выберите настройки качества и формата слайдов
4. Нажмите **«Конвертировать»**
5. Выберите место сохранения PPTX файла
6. Дождитесь завершения процесса

## 📋 Системные требования

- **macOS** 10.15 (Catalina) или новее
- **Python 3.8** или новее (встроен в macOS 12+ или установите с python.org)
- **Зависимости:**
  - PyMuPDF (для работы с PDF)
  - python-pptx (для создания PPTX)
  - Pillow (для обработки изображений)

## 🗑️ Удаление

### Если установлено через DMG:
Просто перетащите приложение из папки **Applications** в **Корзину**.

### Если установлена через скрипт:
```bash
./uninstall.sh
```

## 🛠️ Решение проблем

### Ошибка «Python 3 не найден»
Установите Python 3:
```bash
# Через Homebrew (рекомендуется)
brew install python3

# Или скачайте с официального сайта
# https://www.python.org/downloads/macos/
```

### Ошибка «ModuleNotFoundError: No module named 'PIL'»
Установите недостающие зависимости:
```bash
pip3 install PyMuPDF python-pptx Pillow
```

Или используйте офлайн-установщик:
```bash
# На компьютере с интернетом
./bundle_deps.sh

# Скопируйте папку offline_deps на целевой компьютер
# Затем запустите
python3 offline_installer.py
```

### Приложение не запускается
1. Проверьте, что Python 3 установлен (`python3 --version`)
2. Проверьте права доступа к приложению
3. Попробуйте запустить через терминал:
   ```bash
   /Applications/PDFToPPTXConverter.app/Contents/MacOS/Launcher
   ```

### Ошибка при создании DMG (hdiutil not found)
Команда `hdiutil` доступна только на macOS. Если вы видите эту ошибку:
- На Linux/Windows: используйте созданную структуру `.app` бандла напрямую
- Скопируйте `build/PDFToPPTXConverter.app` в нужное место
- Для запуска используйте: `python3 pdf_to_pptx_gui.py`
