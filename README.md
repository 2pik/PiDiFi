# PDF в PPTX Converter для macOS

Приложение для конвертации PDF файлов в PowerPoint презентации с графическим интерфейсом.

![macOS](https://img.shields.io/badge/macOS-10.15+-silver?logo=apple)
![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)

## Особенности

- ✅ **Работает только локально** — все данные обрабатываются на вашем компьютере
- ✅ **Без облачных сервисов** — никакие файлы не передаются в интернет
- ✅ **Графический интерфейс** в стиле Apple Human Interface Guidelines
- ✅ **Автоматическая тема** — светлая/темная в зависимости от настроек системы
- ✅ **Простая установка** — скачайте DMG и перетащите в Applications
- ✅ **Легкое удаление** — просто перетащите в Корзину или используйте скрипт

## Системные требования

- macOS 10.15 (Catalina) или новее
- Python 3.8 или новее (для установки из исходников)
- ~50 MB свободного места

## Установка

### Вариант 1: Из DMG файла (рекомендуется)

1. Скачайте `PDFToPPTXConverter-Installer.dmg`
2. Откройте DMG файл двойным кликом
3. Перетащите `PDFToPPTXConverter.app` в папку `Applications`
4. Запустите приложение из Applications

### Вариант 2: Из исходников

```bash
chmod +x install.sh
./install.sh
```

## Запуск приложения

### Через Finder:
1. Откройте Finder
2. Перейдите в `Applications` (Программы)
3. Дважды кликните на `PDFToPPTXConverter.app`

### Через Терминал:
```bash
open ~/Applications/PDFToPPTXConverter.app
```

### ⚠️ Первый запуск
macOS может предупредить о неизвестном разработчике:
1. Откройте **Системные настройки** → **Защита и безопасность**
2. Нажмите **Разрешить** рядом с сообщением о блокировке приложения

Или кликните правой кнопкой мыши на приложении → Открыть → Открыть

## Использование

1. Нажмите **"Выбрать PDF"** и укажите файл для конвертации
2. (Опционально) Нажмите **"Изменить"** чтобы выбрать место сохранения PPTX
3. Нажмите **"Конвертировать"**
4. Дождитесь завершения процесса
5. Файл будет сохранен, папка откроется автоматически

## Удаление

### Вариант 1: Через Finder
Просто перетащите `PDFToPPTXConverter.app` из папки Applications в Корзину.

### Вариант 2: Через скрипт
```bash
./uninstall.sh
```

Для полного удаления зависимостей Python (опционально):
```bash
pip3 uninstall customtkinter PyMuPDF python-pptx Pillow
```

## Зависимости

Приложение автоматически устанавливает следующие пакеты при первом запуске:
- `customtkinter` — современный GUI фреймворк
- `PyMuPDF` — обработка PDF файлов
- `python-pptx` — создание PowerPoint презентаций
- `Pillow` — работа с изображениями

## Как это работает

1. Приложение открывает PDF файл с помощью PyMuPDF
2. Каждая страница рендерится в изображение высокого качества (2x масштаб)
3. Изображения добавляются на слайды PowerPoint в формате 16:9
4. Результат сохраняется как .pptx файл

## Безопасность

- Все файлы обрабатываются **только локально** на вашем компьютере
- Никакие данные **не передаются** в интернет
- Временные файлы удаляются сразу после обработки
- Исходный PDF файл не изменяется

## Поддержка форматов

- **Входной формат**: PDF (любые версии)
- **Выходной формат**: PPTX (PowerPoint 2007+)
- **Размер слайда**: 16:9 (стандартный широкоформатный)
- **Качество**: 2x увеличение разрешения для четкости текста

## Troubleshooting

### Приложение не запускается
Убедитесь, что установлен Python 3:
```bash
python3 --version
```

### Ошибка при установке зависимостей
Попробуйте обновить pip:
```bash
pip3 install --upgrade pip
```

### Конвертация зависает на больших файлах
Большие PDF файлы могут обрабатываться несколько минут. Индикатор прогресса показывает статус.

### macOS блокирует приложение
1. Кликните правой кнопкой мыши на приложении
2. Выберите "Открыть"
3. Подтвердите открытие во всплывающем окне
