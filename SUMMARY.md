## 🚀 Как использовать

### Для разработчика (подготовка пакета)

```bash
# 1. На компьютере с интернетом
./bundle_deps.sh

# 2. Проверить содержимое
ls -la offline_deps/

# 3. Упаковать для передачи
zip -r PDFToPPTXConverter-offline.zip \
    *.py *.sh *.md requirements.txt offline_deps/
```

### Для пользователя (установка без интернета)

```bash
# 1. Распаковать архив
unzip PDFToPPTXConverter-offline.zip
cd PDFToPPTXConverter-offline

# 2. Установить зависимости
python3 offline_installer.py

# 3. Запустить приложение
python3 pdf_to_pptx_gui.py

# Или создать DMG
bash build_dmg.sh
```

## 📋 Что входит в офлайн-пакет

### Зависимости (wheel-файлы)
- **PyMuPDF** — обработка PDF файлов
- **python-pptx** — создание PowerPoint презентаций  
- **Pillow** — работа с изображениями

### Поддерживаемые платформы
- ✅ macOS ARM64 (Apple Silicon: M1, M2, M3)
- ✅ macOS x86_64 (Intel Mac)

### Размер пакета
- Wheel-файлы: ~100-150 МБ
- Исходный код: ~50 КБ
- **Итого:** ~100-150 МБ

## 🔧 Технические детали

### bundle_deps.sh
- Скачивает wheel-файлы для обеих архитектур macOS
- Использует `pip download --only-binary=:all:`
- Сохраняет в папку `offline_deps/`

### offline_installer.py
- Проверяет установленные зависимости
- Устанавливает из локальных wheel-файлов
- Работает полностью автономно (`--no-index --find-links`)

### build_dmg.sh (обновленный)
- Проверяет зависимости перед сборкой
- Автоматически запускает offline_installer.py если нужно
- Копирует offline_deps внутрь DMG
- Создает самодостаточный установщик

### Лаунчер приложения
- Проверяет зависимости при каждом запуске
- Пытается установить из встроенных wheel-файлов
- Показывает понятные сообщения об ошибках

## 🎯 Преимущества решения

1. **Полная автономность** — не требуется интернет после подготовки пакета
2. **Простота использования** — одна команда для установки
3. **Универсальность** — работает на Apple Silicon и Intel Mac
4. **Автоматизация** — GitHub Actions собирает пакет при релизе
5. **Надежность** — проверка зависимостей на каждом этапе

## 📞 Решение проблем

| Проблема | Решение |
|----------|---------|
| ModuleNotFoundError | `python3 offline_installer.py` |
| offline_deps не найдена | Запустить `./bundle_deps.sh` с интернетом |
| Python не найден | `brew install python3` |
| DMG не создается | Проверить наличие хитилу: `hdiutil version` |
